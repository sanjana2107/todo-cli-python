# todo-cli-python
# To-Do CLI App  Description A simple command-line To-Do List application built with Python. It allows users to add, view, and delete tasks. Features - Add tasks - View tasks - Delete tasks - Saves tasks using JSON  How to Run 1. Clone the repository 2. Run:    python main.py   Example 1. View Tasks 2. Add Task 3. Delete Task 4. Exit
